# restaurant-mini-system

Restaurant mini system

<br>

## Additional Context

Just a mini system.

### Scope

- Login (user or admin)
- Register (user)
- User
  - Reservation
    - Edit account information
    - Select number of persons
    - Select date and time
  - Generate reservation code
- Admin
  - Reservations
    - Change status to on site
    - Remove reservation
  - Accounts
    - Delete accounts
